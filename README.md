
# ELF/ARM Linker

A linker for ELF/ARM binaries implemented in C — built as part of 
a systems programming course at Université Grenoble Alpes (2025).

## Features
- Symbol resolution between object files
- Section merging (.text, .data, .bss)
- Relocation handling for ARM architecture

## Technologies
- Language: C
- Platform: Linux
